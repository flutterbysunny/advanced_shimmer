# Advanced Shimmer

A customizable Flutter shimmer loading package with multiple shimmer animation styles.

## Features

- Linear shimmer
- Wave shimmer
- Pulse shimmer
- Glow shimmer
- Diagonal shimmer
- Border glow shimmer

Includes ready-made widgets:

- `AdvancedShimmer`
- `ShimmerCard`
- `ShimmerText`
- `ShimmerAvatar`
- `ShimmerListTile`

---

## Installation

Add to `pubspec.yaml`:

```yaml
dependencies:
  advanced_shimmer: ^1.0.0
